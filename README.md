This project implements a logical deduction system in Prolog using Wang's algorithm to determine whether a conclusion can be deerived from a given set of premises. The program represents logical formulas with custom operators and systematically applies both non-branching and branching inference rules (such as negation, conjunction, disjunction, and implication) to transform and evaluate sequents. By recursively reducing expressions, the system either proves the conclusion or provides a counterexample interpretation when the conclusion does not follow from the premises.
